# 前言

欢迎来到基于SSM的勤工助学管理系统项目！该项目旨在帮助高校管理勤工助学活动，提高工作效率，方便学生与管理工作。以下是对该项目的详细介绍。

# 内容介绍

本项目采用Java语言，结合Spring、Spring MVC和MyBatis框架，构建了一套完善的勤工助学管理系统。系统主要包括以下功能模块：学生信息管理、岗位信息管理、申请与审批、工资发放等。通过该项目，学生可以在线查看岗位信息、提交申请，管理员可以发布岗位、审批申请、管理学生信息等，实现勤工助学的便捷管理。

# 技术介绍

## 语言：Java
## 使用框架：Spring、Spring MVC、MyBatis
## 前端技术：JS、Vue、CSS3

## 前端环境：Node.Js 12\14\16
